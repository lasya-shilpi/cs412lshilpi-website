F5-Advanced Search Project

Automated a specific feature to help employees construct an advanced search framework. We utilized javascript libraries such as react and redux to build the front-end framework and python libraries such as Django to build API requests to switch between 3 different searches: Basic Search, Logical Search, and Advanced Search. The searches had various functionalities; we first implemented the Basic Search, which let the employees carry out simple searches within their database. The Logical Search function allowed employees to search many fields at once. Next, we implemented an Advanced Search, which allowed employees to filter the search based on a SQL command.
